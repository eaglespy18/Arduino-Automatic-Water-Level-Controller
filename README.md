# Automatic Water Level Controller

## Project Description
This project monitors water levels (Low, Medium, High) using a water sensor and Arduino UNO.

## Components Used
- Arduino UNO
- Water Level Sensor
- 3 LEDs (Red, Yellow, Green)
- 3 × 220Ω resistors
- Piezo Buzzer
- Breadboard and jumper wires

## How It Works
- Red LED + buzzer → Low water
- Yellow LED → Medium water
- Green LED → High water
